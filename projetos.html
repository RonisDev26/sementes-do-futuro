3. Calculadora de IMC

Data: Março 2026

Tipo:Algoritimo/Pseudocódigo

Descrição

Algoritimo completo para calcular o Índice de Massa Corporal, incluindo validações, classificação e opção de repetir o cálculo.

Funcionalidades

  • Entrada de peso e altura
  • Cálculo do IMC (peso ÷ altura²)
  • Classificação segundo tabela da OMS
  • validação de dados inválidos
  • Loop para multiplos cálculos

Estruturas utilizadas

  • Sequência: Passos organizados
  • Decisão: Classificação do IMC
  • Repetição: Opção de caluclar novamente
  • Variáveis: Armazenamento de dados
Ver pseudocódigo completo 

                        1.INICIO
                        2.peso, altura, imc, continuar ⬅️ VARIÁVEIS
                        3.Repita
                          3.1. Mostrar ''=== CALCULADORA DE IMC ===''
                          3.2. Perguntar ''Digite o seu peso (kg):''
                          3.3. LER peso
                          3.4. SE peso <= 0 ENTÃO
                               3.4.1. Mostrar ''Peso inválido!''
                               3.4.2. VOLTAR para 3.2
                          3.5. Perguntar ''Digite sua altura (m):''
                          3.6. LER altura
                          3.7. SE altura <= 0 ENTÃO
                               3.7.1. Mostrar ''Altura inválida''
                               3.7.2. VOLTAR para 3.5
                          3.8. imc ← peso ÷ (altura x altura)
                          3.9. Mostrar ''Seu IMC é: '' +imc
                          3.10. SE imc < 18.5 ENTÃO
                                Mostrar ''Calssifação: Abaixo do peso''
                          3.11. SENÃO SE imc < 25 ENTÃO
                                Mostrar ''Classificação: Peso normal''
                          3.12. SENÃO SE imc < 30 ENTÃO
                                Mostrar ''Classificação: Sobrepeso''
                          3.13. SENÃO
                                Mostrar ''Classificação: Obesidade''
                          3.14. Perguntar ''Calcular para outra pessoa? (S/N)
                          3.15. LER continuar
                        4. ATÉ continuar = ''N''
                        5. Mostrar ''Obrigado por usar a calculadora!''
                        6. FIM
